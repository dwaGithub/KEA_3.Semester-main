This shows how to make a real REST API

HTTP Method       Endpoint
GET               /dinosaurs         Get ALL dinosaurs
GET               /dinosaurs/{id}    Get dinosaur by id
POST              /dinosaurs         Create a dinosaur
PUT               /dinosaurs/{id}    Update a dinosaur by id
PATCH             /dinosaurs/{id}    Update some values on dinosaur by id
DELETE            /dinosaurs/{id}    Delete a dinosaur by id
